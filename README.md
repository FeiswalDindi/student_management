# KCA Student Management System

![PHP](https://img.shields.io/badge/PHP-777BB4?style=for-the-badge&logo=php&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=for-the-badge&logo=mysql&logoColor=white)
![HTML/CSS](https://img.shields.io/badge/HTML5_&_CSS3-E34F26?style=for-the-badge&logo=html5&logoColor=white)

A full-stack web application for managing university student records, course catalogs, and academic performance.

---

## Features

- **Full CRUD Operations** - Create, read, update, and delete student records
- **Course Management** - Add and manage course offerings
- **Academic Records** - Map students to courses with grade tracking
- **Responsive Dashboard** - Custom Flexbox layout with institutional styling
- **Modular Architecture** - Reusable components (header, sidebar, footer)

---

## Tech Stack

| Technology | Purpose |
|------------|---------|
| PHP | Backend logic and data handling |
| MySQL | Database management |
| HTML/CSS | Frontend structure and styling |
| XAMPP | Local development environment |
